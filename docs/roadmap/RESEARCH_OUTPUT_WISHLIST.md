# Research Output Wishlist - 100 Missing Data Items

*Analysis based on Tigo Paraguay report output - December 2025*

This document identifies gaps in current research output and provides actionable sources for improvement.

---

## Overview

**Current State:** Reports generate ~50-70% quality scores with significant data gaps in:
- Financial metrics (subsidiary-level data)
- Operational KPIs (employees, infrastructure)
- Leadership details (executive team)
- Verified market data (regulator sources)

**Goal:** Achieve 85%+ quality scores by integrating additional data sources.

## Implementation Recommendations

### Phase 1: Quick Wins (FREE sources)
1. **LinkedIn scraping** for employee count, executive names
2. **Company website** extraction for locations, products
3. **Wikipedia** structured data extraction
4. **Regulator websites** for official statistics
5. **Press release** aggregation for recent news

### Phase 2: API Integrations
1. **Ookla/OpenSignal** for network quality metrics
2. **Google Maps API** for location/store data
3. **News APIs** (Google News, NewsAPI) for sentiment
4. **SEC EDGAR** for US-listed parent companies

### Phase 3: Premium Data
1. **GSMA Intelligence** subscription for telecom KPIs
2. **Capital IQ/Bloomberg** for financial data
3. **LinkedIn Sales Navigator** for org charts
4. **Sustainalytics/MSCI** for ESG scores

## Suggested New Agents/Modules

Based on this analysis, consider implementing:

1. **LinkedInScraperAgent** - Extract employee counts, executive names, org structure
2. **RegulatorDataAgent** - Fetch official statistics from telecom regulators
3. **ParentCompanyAgent** - Extract subsidiary data from parent company filings
4. **NetworkQualityAgent** - Integrate Ookla/OpenSignal APIs
5. **CentralBankAgent** - Fetch fintech/mobile money statistics
6. **ProcurementAgent** - Search government contract databases
7. **CompanyRegistryAgent** - Extract legal entity data, board composition
